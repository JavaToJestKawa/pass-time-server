# Pass Time Server

Client-server application for concurrent time calculations using Java sockets and YAML configuration.

## Features

* Server-client communication using Java sockets
* Concurrent and sequential client execution
* YAML-based configuration
* Time interval calculations in calendar notation
* Per-client and server-side logging

## Technologies

* Java
* Java NIO
* SnakeYAML

## How to run

Compile the project and run the main class with SnakeYAML in the classpath
